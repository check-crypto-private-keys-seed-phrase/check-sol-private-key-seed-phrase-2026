## ❓ Common Questions

**Does this tool steal my private keys?**  
No. It runs completely offline. Your keys never leave your computer. No internet connection is used for validation.

**Can this hack Solana wallets?**  
No. This tool only checks if a key is correctly formatted. It does not and cannot check if a key has a balance.

**What's the difference between Base58 and byte array format?**  
Base58 is the standard human-readable Solana private key format. Byte array (JSON) is what Phantom and Solflare export. Both are supported.

**Does it support 12-word seed phrases?**  
Yes. BIP39 mnemonics with 12, 18, or 24 words are all supported.

**Will it show me the Solana address?**  
Yes. For valid private keys, the tool derives and displays the corresponding Solana public key address.

**What if the key is valid but has no balance?**  
The tool only validates formatting. It does not connect to any blockchain to check balances.

**Is this safe to use?**  
Yes. It's open-source and runs offline. You can disconnect your internet before using it for absolute peace of mind.
